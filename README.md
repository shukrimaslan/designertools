# Designkit 🎨

**Designkit** is a curated collection of 16 micro-tools, utilities, and creative workflows built for designers, UI/UX professionals, and digital creators. No login, no ads, no friction — just instant design utilities.

🔗 **[Live Demo](https://shukrimaslan.github.io/designertools/)**

---

## 🛠 Features

The toolkit includes 16 essential micro-apps categorized for various design needs:

### Color & Type
- **Contrast Checker**: Instantly get WCAG AA/AAA pass/fail results for color pairs.
- **Palette Generator**: Auto-generate a full design system ramp from a single seed color.
- **Type Scale Calculator**: Pick a base size and ratio to preview a modular type scale.
- **Font Pairing Preview**: Preview heading and body Google Font combinations with live text.

### Layout
- **Spacing Visualizer**: Input a base unit to preview your 4pt / 8pt spacing grid.
- **Grid Builder**: Set columns, gutter, and margin to export CSS Grid or Tailwind code.
- **Shadow Generator**: Tweak offset, blur, and spread with a live preview and CSS export.
- **Aspect Ratio Tool**: Lock a ratio and calculate missing width or height dimensions.

### Assets & Copy
- **Dummy Text Generator**: Generate realistic UI copy (names, bios, emails) instead of Lorem Ipsum.
- **SVG Cleaner**: Paste SVG markup to strip bloat and get an optimized version.
- **Icon Picker**: Search common UI icons and copy SVG or React component code.
- **Gradient Generator**: Build linear or radial gradients visually and copy the CSS.

### Handoff
- **Hex → Token Converter**: Convert hex values to CSS custom properties, JSON, or Tailwind config.
- **Rem / px Converter**: Quick conversion between pixels and rem units based on a base font size.
- **Breakpoint Visualizer**: See how layouts shift across mobile, tablet, and desktop views.
- **Easing Previewer**: Edit cubic-bezier curves and preview motion on sample elements.

## 🚀 Getting Started

Since Designkit is built as a static web application, there's no complex setup required.

1.  Clone the repository:
    ```bash
    git clone https://github.com/shukrimaslan/designertools.git
    ```
2.  Open `index.html` in any modern web browser.

---

## 💻 Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flexbox, and CSS Grid for layout and styling.
- **Vanilla JavaScript**: Pure JS for all tool logic and interactivity (no frameworks).
- **Google Fonts**: DM Serif Display, DM Mono, and Sora.
